# 登录链路与监控调度修复 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复登录卡片刷新失败、视频号 QR 截图兜底、登录不复用 tab、快手未登录调度四类问题。

**Architecture:** 三账号共用浏览器窗口导致卡片登录与监控并发争抢 CDP。修复分三层：(1) 浏览器会话复用——`ensureLoginTab` 优先复用监控已开的平台 tab、`getBrowser` CDP 握手失败重试、`sendLoginQR` windowId 改用 externalId；(2) 未登录调度——快手 `handleLogin` 补 `login_required` 状态切换（probe 状态机已在 `getAllActiveUsers` 正常工作），前端卡片加"恢复登录"按钮 + 后端 `/probe-login` 端点；(3) QR 捕获——视频号 qrSelectors 改 `img.qrcode`、qrRefreshSelector 改 `.refresh-wrap`。

**Tech Stack:** TypeScript / tsx / Jest / ts-jest（ts-api-gateway），React / Next.js（admin-dashboard），patchright CDP（browser-core），Prisma，Redis，BitBrowser/RoxyBrowser API。

```typescript
  // 2. 未找到则优先复用监控已打开的同域名平台 tab（不新建连接/tab）
  if (!record) {

            if (!url.includes(config.domain)) continue;
```

```typescript
/** POST /api/v1/matrix/monitor/accounts/:userId/probe-login — 立即探测登录态（绕过 cooldown） */
router.post('/monitor/accounts/:userId/probe-login', async (req: Request, res: Response) => {
  try {
    const userId = parseInt(req.params.userId, 10);
    if (isNaN(userId)) return res.status(400).json({ error: '无效的 userId' });

    const { prisma } = await import('../lib/prisma');
    const account = await prisma.platformAccount.findUnique({
      where: { id: userId },
      select: { platform: true, window: { select: { externalId: true } } },
    });
    if (!account) return res.status(404).json({ error: '未找到账号' });
    const windowId = account.window?.externalId ? String(account.window.externalId) : '';
    if (!windowId) return res.status(400).json({ error: '账号未关联浏览器窗口' });

    const { triggerLoginProbe } = await import('../services/monitorService');
    // force=true 立即探测，绕过 30min cooldown
    await triggerLoginProbe(userId, account.platform, windowId, undefined, true);
    res.json({ message: '已触发登录探测', userId, platform: account.platform });
  } catch (err: any) {
    logger.error({ err: err.message }, '[probe-login] 探测失败');
    res.status(500).json({ error: err.message || '探测失败' });
  }
});
```
