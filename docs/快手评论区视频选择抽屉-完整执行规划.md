## 四、安全点击策略

### 策略 A：点击 .date 元素（推荐，已验证）

```javascript
const target = document.querySelectorAll('.drawer.video-list .video-item')[N];
const dateEl = target.querySelector('.video-info__content__date');
dateEl.dispatchEvent(new MouseEvent('click', { bubbles: true }));
```

### 策略 B：坐标级点击空白区（最稳健，模拟 CDP）

```javascript
const target = document.querySelectorAll('.drawer.video-list .video-item')[N];
const content = target.querySelector('.video-info__content');
const rect = content.getBoundingClientRect();

// 空白区中心：日期底(rect.top+49) 到 detail顶(rect.top+78)，取中间 rect.top+63
const x = rect.left + rect.width / 2;   // 水平居中 ≈ rect.left + 90
const y = rect.top + 63;                // 空白区垂直中心

const el = document.elementFromPoint(x, y);
el.dispatchEvent(new MouseEvent('click', {
  bubbles: true,
  clientX: x,
  clientY: y
}));
```

### 策略 C：CDP Input.dispatchMouseEvent（纯协议级）

```
CDP Runtime.evaluate → 获取 .video-info__content__date 的 getBoundingClientRect()
                     → 计算中心点 (x, y)
CDP Input.dispatchMouseEvent → type:"mousePressed" x y button:"left" clickCount:1
CDP Input.dispatchMouseEvent → type:"mouseReleased" x y button:"left" clickCount:1
```

## 五、完整操作流程

```

Step 2: 列出所有视频卡片，找到目标
  const items = drawer.querySelectorAll('.video-item');
  items.forEach((item, i) => {
    const title = item.querySelector('.video-info__content__title').textContent.trim();
    const isActive = item.classList.contains('is-active');
    // 选中第一个 !isActive 的作为目标
  });

Step 3: 获取安全点击坐标
  const targetItem = items[targetIndex];
  const dateEl = targetItem.querySelector('.video-info__content__date');
  const rect = dateEl.getBoundingClientRect();
  const clickX = rect.left + rect.width / 2;
  const clickY = rect.top + rect.height / 2;

Step 4: 执行点击
  方式 1: elementFromPoint(clickX, clickY).dispatchEvent(MouseEvent)
  方式 2: page.mouse.click(clickX, clickY)  （Playwright）
  方式 3: CDP Input.dispatchMouseEvent        （纯协议）

Step 5: 等待并验证
  等待 1.5 秒
  targetItem.classList.contains('is-active') === true
  旧卡片 is-active === false
  页面主体评论列表已切换
```

## 九、注意事项

1. **事件冒泡必须开启**：`{ bubbles: true }`，否则 Vue 组件收不到事件
2. **抽屉选中后自动关闭**：点击成功后抽屉会关闭，需要后续操作需重新打开
3. **作用域限定**：所有选择器必须以 `.drawer.video-list` 开头，防止点到页面主体区域
4. **等待选中反馈**：必须等待 `.video-item.is-active` 出现才算成功，不能仅凭点击完成判断
5. **不使用 preventsDefault**：卡片不是链接，阻止默认行为无意义且可能破坏 Vue 事件
