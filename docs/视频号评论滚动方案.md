## 视频号评论管理页面 — Playwright 自动化采集方案

### 一、页面架构与核心技术难点

目标页面 `https://channels.weixin.qq.com/platform/interaction/comment` 使用 **wujie 微前端** 架构，评论管理的核心 UI 运行在 Shadow DOM 内，外层 `document` 上无法直接查询到任何业务元素。

**核心穿透方式：**

```typescript
// Playwright 中通过 locator 穿透 Shadow DOM
// wujie-app 是宿主元素，其 shadowRoot 内包含完整页面
const shadowHost = page.locator('wujie-app');
// Playwright 默认自动穿透 open 模式的 Shadow DOM，可直接用 CSS 选择器
```

> **Playwright 优势：** Playwright 默认支持 Shadow DOM 穿透（`locator` 会自动搜索所有 shadow roots），无需手动获取 `shadowRoot`。这是与原生 JS 操作最大的区别。

**页面布局：**

```

│  左侧导航栏 (~200px)  │       主内容区 (~1224px)       │
```

### 三、懒加载机制详解

#### 3.1 滚动懒加载（视频列表 & 评论列表共用模式）

页面使用 **IntersectionObserver** 监听底部哨兵元素 `.scroll-bottom__tips`。当哨兵进入视口时触发数据加载。

**关键发现：只有真实鼠标滚轮事件能触发懒加载。**

| 方式 | Playwright 对应 API | 是否有效 |
|------|---------------------|---------|
| 真实鼠标滚轮 | `page.mouse.wheel(0, deltaY)` | ✅ **推荐** |
| `el.scrollTo()` | `page.evaluate(() => el.scrollTo(...))` | ⚠️ 可能有效但不稳定 |
| 直接设 scrollTop | `page.evaluate(() => el.scrollTop = val)` | ❌ 无效 |
| scrollTop + dispatchEvent | `page.evaluate(...)` | ❌ 无效 |

#### 3.2 "展开更多回复"按钮（子评论懒加载）

当某条根评论的子回复数超过初始展示量（3条）时，`.comment-main-content` 底部会出现 `.load-more` 按钮：

```html
<div class="load-more">
  <div class="load-more__btn">
    <span>展开更多回复</span>
    <span class="weui-icon-outlined-arrow"></span>
  </div>
</div>
```

- **触发方式：** 点击 `.load-more__btn`（普通 click 事件即可）
- **加载行为：** 每次点击触发 `comment_list` API，加载约 10 条子回复
- **终止条件：** `.load-more` 元素从 DOM 中消失 = 所有回复已加载完毕
- **实测数据：** 22条子回复 → 初始显示3条 → 第1次点击→13条 → 第2次点击→22条 → 按钮消失

#### 4.2 Shadow DOM 穿透策略

```typescript
/**
 * 获取 Shadow DOM 内的元素
 * Playwright locator 默认穿透 Shadow DOM，直接用 CSS 选择器即可
 */
function shadowLocator(page: Page, selector: string): Locator {
  return page.locator(selector);
}

/**
 * 在 Shadow DOM 内执行 JS（evaluate 也能穿透 Shadow DOM）
 */
async function evalInShadow<T>(page: Page, fn: string): Promise<T> {
  return page.evaluate(fn) as Promise<T>;
}
```

#### 4.3 容器定位与鼠标移动

```typescript
/**
 * 获取滚动容器的可视区域坐标（用于 mouse.wheel 定位）
 */
async function getContainerCenter(page: Page, selector: string) {
  const rect = await page.locator(selector).boundingBox();
  if (!rect) throw new Error(`容器 ${selector} 未找到`);
  return {
    x: rect.x + rect.width / 2,
    y: rect.y + rect.height / 2,
    rect
  };
}

/**
 * 将鼠标移动到指定容器区域并模拟滚动
 * @param page - Playwright Page
 * @param selector - 滚动容器的 CSS 选择器
 * @param deltaY - 滚动像素量（正值向下）
 */
async function scrollInContainer(page: Page, selector: string, deltaY: number = 300) {
  const { x, y } = await getContainerCenter(page, selector);

  // 1. 移动鼠标到容器中心
  await page.mouse.move(x, y);

  // 2. 模拟鼠标滚轮（真实 wheel 事件，能触发 IntersectionObserver）
  await page.mouse.wheel(0, deltaY);
}

/**
 * 滚动到容器底部（持续滚动直到 scrollHeight 不再增长）
 */
async function scrollToBottom(
  page: Page,
  containerSelector: string,
  scrollStep: number = 300,
  delayMs: number = SCROLL_DELAY_MS
) {
  let prevScrollHeight = 0;
  let retries = 0;

  while (retries < MAX_SCROLL_RETRIES) {
    // 执行滚动
    await scrollInContainer(page, containerSelector, scrollStep);
    await page.waitForTimeout(delayMs);

    // 获取当前滚动状态
    const scrollState = await evalInShadow<{ scrollHeight: number; scrollTop: number; clientHeight: number }>(
      page,
      `(() => {
        const el = document.querySelector('wujie-app').shadowRoot
          .querySelector('${containerSelector}');
        return { scrollHeight: el.scrollHeight, scrollTop: el.scrollTop, clientHeight: el.clientHeight };
      })()`
    );

    if (scrollState.scrollHeight === prevScrollHeight) {
      retries++;
    } else {
      retries = 0;
      prevScrollHeight = scrollState.scrollHeight;
    }
  }
}
```

```typescript
/**
 * 在左侧视频列表中滚动查找目标视频
 */
async function findAndClickVideo(page: Page, videoName: string) {
  const feedContainerSelector = '.feeds-container';

  // 循环：滚动 → 查找 → 找到则点击
  let found = false;
  let scrollRetries = 0;

  while (!found && scrollRetries < 20) {
    // 在 Shadow DOM 中查找目标视频
    found = await evalInShadow<boolean>(page, `(() => {
      const body = document.querySelector('wujie-app').shadowRoot.querySelector('body');
      const feeds = body.querySelectorAll('.comment-feed-wrap');
      return Array.from(feeds).some(f => {
        const title = f.querySelector('.feed-title');
        return title && title.textContent.trim().includes('${videoName}');
      });
    })()`);

    if (found) break;

    // 未找到，向下滚动加载更多
    await scrollInContainer(page, feedContainerSelector, 500);
    await page.waitForTimeout(SCROLL_DELAY_MS);
    scrollRetries++;
  }

  if (!found) throw new Error(`视频 "${videoName}" 未在列表中找到`);

  // 获取目标视频的位置并点击
  const rect = await evalInShadow<{ x: number; y: number; w: number; h: number }>(
    page,
    `(() => {
      const body = document.querySelector('wujie-app').shadowRoot.querySelector('body');
      const feeds = body.querySelectorAll('.comment-feed-wrap');
      const target = Array.from(feeds).find(f => {
        const title = f.querySelector('.feed-title');
        return title && title.textContent.trim().includes('${videoName}');
      });
      if (!target) return null;
      const r = target.getBoundingClientRect();
      return { x: r.x, y: r.y, w: r.width, h: r.height };
    })()`
  );

  // 如果目标不在可视区域，先滚动到可见
  if (rect.y < 300 || rect.y > 800) {
    await evalInShadow(page, `(() => {
      const body = document.querySelector('wujie-app').shadowRoot.querySelector('body');
      const container = body.querySelector('.feeds-container');
      const feeds = body.querySelectorAll('.comment-feed-wrap');
      const target = Array.from(feeds).find(f => {
        const title = f.querySelector('.feed-title');
        return title && title.textContent.trim().includes('${videoName}');
      });
      if (target) {
        target.scrollIntoView({ block: 'center' });
      }
    })()`);
    await page.waitForTimeout(500);
  }

  // 点击目标视频
  await page.mouse.click(rect.x + rect.w / 2, rect.y + rect.h / 2);
  await page.waitForTimeout(2000);

  console.log(`✅ 已点击视频 "${videoName}"`);
}
```

```typescript
/**
 * 滚动右侧评论面板，加载所有根评论

async function loadAllRootComments(page: Page) {
  const commentContainerSelector = '.feed-comment__wrp';
  let prevCount = 0;
  let stableCount = 0;

  while (stableCount < MAX_SCROLL_RETRIES) {
    // 向下滚动
    await scrollInContainer(page, commentContainerSelector, 300);
    await page.waitForTimeout(SCROLL_DELAY_MS);

    // 统计当前根评论数
    const count = await evalInShadow<number>(page, `(() => {
      const body = document.querySelector('wujie-app').shadowRoot.querySelector('body');
      const commentWrap = body.querySelector('.feed-comment__wrp');
      const innerDiv = commentWrap.querySelector('.scroll-list > div');
      return innerDiv ? innerDiv.querySelectorAll(':scope > .comment-item').length : 0;
    })()`);

    if (count === prevCount) {
      stableCount++;
    } else {
      stableCount = 0;
      prevCount = count;
    }

    console.log(`  评论加载中... 当前 ${count} 条根评论`);
  }

  console.log(`✅ 根评论全部加载: ${prevCount} 条`);
  return prevCount;
}
```

#### 4.8 步骤五：采集所有评论数据（根评论 + 子评论拼接）

```typescript
interface Comment {
  username: string;
  isAuthor: boolean;
  time: string;
  content: string;
  likes: string;
  replies: Comment[];  // 子回复
}

/**
 * 采集所有评论（含子回复），拼接为树形结构
 */
async function collectAllComments(page: Page): Promise<Comment[]> {
  const rawData = await evalInShadow<any>(page, `(() => {
    const body = document.querySelector('wujie-app').shadowRoot.querySelector('body');
    const commentWrap = body.querySelector('.feed-comment__wrp');
    const innerDiv = commentWrap.querySelector('.scroll-list > div');
    const rootItems = innerDiv.querySelectorAll(':scope > .comment-item');

    function parseComment(item) {
      const main = item.querySelector('.comment-item-main');
      if (!main) return null;

      const username = main.querySelector('.comment-user-name')?.textContent?.trim() || '';
      const isAuthor = !!main.querySelector('.comment-author-bandage');
      const time = main.querySelector('.comment-time')?.textContent?.trim() || '';
      const content = main.querySelector('.comment-content')?.textContent?.trim() || '';
      const likeEl = main.querySelector('.like-action');
      const likes = likeEl?.textContent?.trim() || '0';

      // 解析子回复
      const replyList = main.querySelector('.comment-reply-list');
      const replies = [];
      if (replyList) {
        const replyItems = replyList.querySelectorAll(':scope > .comment-item');
        replyItems.forEach(replyItem => {
          const reply = parseComment(replyItem);
          if (reply) replies.push(reply);
        });
      }

      return { username, isAuthor, time, content, likes, replies };
    }

    const comments = [];
    rootItems.forEach(item => {
      const comment = parseComment(item);
      if (comment) comments.push(comment);
    });

    return comments;
  })()`);

  return rawData as Comment[];
}

/**
 * 将评论树展平为列表（根评论 + 子评论拼接在一起）
 */
function flattenComments(comments: Comment[], parentId: string = ''): Array<Comment & { parentId: string; depth: number }> {
  const result: Array<Comment & { parentId: string; depth: number }> = [];

  comments.forEach((c, i) => {
    const id = parentId ? `${parentId}.${i}` : `${i}`;
    result.push({
      ...c,
      parentId: parentId || 'root',
      depth: parentId ? 1 : 0,
      replies: []  // 展平后清空嵌套，子回复单独列出
    });

    // 递归展开子回复
    if (c.replies && c.replies.length > 0) {
      c.replies.forEach((r, j) => {
        result.push({
          ...r,
          parentId: id,
          depth: 1,
          replies: []
        });
      });
    }
  });

  return result;
}
```

```typescript
async function main() {
  const browser = await chromium.launch({
    headless: false,  // 建议先用有头模式调试
    // userDataDir: './browser-data',  // 保持登录态
  });

  const context = await browser.newContext({

  const page = await context.newPage();

  try {
    // === 步骤1: 打开评论管理页面 ===
    await openCommentPage(page);

    const allResults: Record<string, Comment[]> = {};

    for (const videoName of TARGET_VIDEOS) {
      console.log(`\n🎬 开始采集视频: ${videoName}`);

      // === 步骤2: 查找并点击目标视频 ===
      await findAndClickVideo(page, videoName);

      // === 步骤3: 滚动加载所有根评论 ===

      // === 步骤4: 展开所有子回复 ===
      await expandAllReplies(page);

      // === 步骤5: 采集评论数据 ===
      const comments = await collectAllComments(page);
      allResults[videoName] = comments;

      const totalReplies = comments.reduce((sum, c) => sum + c.replies.length, 0);
      console.log(`📊 ${videoName}: ${comments.length} 条根评论, ${totalReplies} 条子回复`);
    }

    // 输出结果
    console.log('\n📋 采集结果:');
    for (const [video, comments] of Object.entries(allResults)) {
      const flat = flattenComments(comments);
      console.log(`\n--- ${video} (${flat.length} 条评论) ---`);
      flat.forEach(c => {
        const prefix = c.depth === 0 ? '💬' : '  ↳';
        const authorTag = c.isAuthor ? ' [作者]' : '';
        console.log(`${prefix} ${c.username}${authorTag} (${c.time}): ${c.content}`);
      });
    }

  } finally {
    await browser.close();
  }
}

main().catch(console.error);
```

```typescript
      console.log(`📤 请求: ${url}`);
      if (request.postData()) {
        console.log(`   POST body: ${request.postData().substring(0, 200)}`);
      }
    }
  });
}
```

### 八、注意事项与风险

1. **Shadow DOM 穿透**：Playwright 的 `locator` 默认穿透 Shadow DOM，但 `page.evaluate` 中需手动获取 `shadowRoot`。

2. **滚动必须用 `mouse.wheel`**：不能用 `page.evaluate` 设置 `scrollTop`，这会跳过 IntersectionObserver 回调。正确方式：
   ```typescript
   await page.mouse.move(x, y);    // 先移动到容器内
   await page.mouse.wheel(0, 300); // 再触发真实滚轮事件
   ```

3. **滚动步长控制**：每次 `wheel` 的 deltaY 建议 200~400px，过大可能跳过哨兵元素触发区域。

4. **等待时间**：每次滚动后等待 1~2 秒，给 API 响应和 DOM 渲染留出时间。

5. **"展开更多回复"按钮位置**：按钮可能在视口之外，需先 `scrollIntoView` 或滚动到可见再点击。

6. **评论数量少时无需滚动**：根评论 ≤ 10 条或子回复 ≤ 3 条时，初始已全部加载。

7. **`scrollIntoView` 的局限性**：在 Shadow DOM 内调用 `scrollIntoView` 通常有效，但如果遇到问题，可改为手动计算 scrollTop。

8. **并发风险**：不要同时操作多个视频的评论加载，页面可能状态混乱。建议逐个视频串行采集。

9. **频率控制**：建议两次 API 请求间隔 ≥ 1.5 秒，避免触发微信反爬机制。
